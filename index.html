EMOTION TEXT AND SPEECH RECOGNITION

In this project, we describe a thorough method for recognizing emotions in text using Python and cutting-edge machine learning algorithms. The combination of NLP with ML enables users to extract insightful information from textual data. The project demonstrates the use of several methods, including feature extraction and text pre-processing powered by NLTK, For improved emotion identification, the system also has a deep learning component built on PyTorch. The project further increases its functionality by adding a user-friendly graphical interface made with PyQt5.
